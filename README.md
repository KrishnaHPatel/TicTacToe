# Tic Tac Toe Game

This is a simple Tic Tac Toe game implemented in C using the GTK+ 3 library. 

## Prerequisites

- GTK+ 3.0: Ensure you have GTK+ 3.0 installed on your system. 

## Compilation and Running

1. **Clone the Repository**:
    ```sh
    git clone https://github.com/your-username/tictactoe.git
    cd tictactoe
    ```

2. **Compile the Program**:
    ```sh
    gcc -o tictactoe tictactoe.c `pkg-config --cflags --libs gtk+-3.0`
    ```

3. **Run the Program**:
    ```sh
    ./tictactoe
    ```

## Files

- `tictactoe.c`: Source code for the Tic Tac Toe game.
- `style.css`: CSS file for styling the buttons
